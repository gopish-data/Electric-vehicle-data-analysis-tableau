# Electric Vehicle Data Analysis Dashboard (Tableau)

## About This Project
This project is a Tableau dashboard created to analyze electric vehicle (EV) data and understand overall vehicle distribution, electric range, and eligibility insights.

The main objective of this project is to study EV growth trends, compare BEV and PHEV vehicles, and analyze state-wise and manufacturer-wise performance.

## What I Did in This Project
- Imported EV dataset into Tableau
- Cleaned and prepared the data
- Created calculated fields for KPIs
- Designed an interactive dashboard with filters
- Built multiple visualizations for comparison and trend analysis

## Key KPIs Tracked
- Total Vehicles
- Average Electric Range
- Total BEV Vehicles
- Total PHEV Vehicles
- % Contribution of BEV and PHEV

## Dashboard Features
- Total Vehicles by Model Year (Trend Analysis)
- Total Vehicles by State (Map View)
- Top 6 Vehicle Manufacturers
- CAFV Eligibility Distribution
- Total Vehicles by Model (Detailed Table)
- Interactive Filters:
  - CAFV Eligibility
  - EV Type
  - Model
  - State

## Insights Observed
- PHEV vehicles contribute a higher percentage compared to BEV
- Certain states have significantly higher EV adoption
- Some manufacturers dominate total EV production
- EV growth trend increases across model years
